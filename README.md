Code for the looped mathematical animations seen on https://wapu.uber.space

Requirements to be able to run everything:
    Python 2.7+ with packages:
        - numpy
        - pillow
        - gizeh
        - moviepy
        - ortools
        - shapely
    ffmpeg
    concorde TSP solver
