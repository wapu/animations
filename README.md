Code for the looped mathematical animations seen on https://wapu.uber.space or Instagram @wapu90

Requirements to be able to run everything:
- Python 3.10+ with packages:
  - numpy
  - scipy
  - fast-tsp
  - shapely
  - pillow
  - gizeh
  - moviepy
- ffmpeg
